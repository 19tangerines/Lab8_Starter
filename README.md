### How are graceful degradation and service workers related?

Graceful degredation is the design practice of making sure an application/software continues to function even if some features or resources are unavailable or limited, like internet speed or connectivity. This is opposed to only being operational when all conditions are perfect and simply failing otherwise. Service workers are one tool to support graceful degradation by caching resources (like the recipe URLs) and serving them when there is low/no connectivity, allowing a website to be partially functional offline instead of completely breaking.
